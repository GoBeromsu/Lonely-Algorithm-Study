## Get It Start
1. [Lonely-Algorithm-Study]를 fork 한다
2. 각자 fork한 Repo에서 자신의 이름으로 폴더를 만든다
3. 푼 문제를 [Lonely-Algorithm-Study]의 Main 브랜치로 PR 날린다

## PR 규칙
- PR 이름 :  [자신의 이름]
- Merge는 일주일에 한 번합니다!
## file 이름 예시
- 프로그래머스 : Programmers_[문제 번호]
- 백준 : Baekjoon-[문제 번호]
